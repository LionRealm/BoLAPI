    Introduction

    Welcome to the BoL Studio API ! This documentation is made by hands. If you want to add more documentation look at the github project. The documentation is not full at the moment.

    You can view code examples in the dark area to the right.

    CallBack Functions

    OnLoad

    • Called when a script is loaded.
    function OnLoad()
    print("OnLoad works")
end

    OnUnload

    • Called when a script is unloaded.
    function OnUnload()
    print("OnUnload works")
end

    OnDraw

    • Called every draw tick.
    • Should be use only for drawing purpose.
    function OnDraw()
    print("OnDraw works")
end

    OnReset

    • Called every draw reset.
    function OnReset()
    print("OnReset works")
end

    OnTick

    • Called every game tick.
    function OnTick()
    print("OnTick works")
end

    OnCreateObj

    • Called when an object is created.
    function OnCreateObj(object)
    print("OnCreateObj works")
end

    OnDeleteObj

    • Called when an object is deleted.
    function OnDeleteObj(object)
    print("OnDeleteObj works")
end

    OnWndMsg

    • Called when the user use the mouse or the keyboard.
    function OnWndMsg(msg, key)
    print("OnWndMsg works")
end

    OnProcessSpell

    • Called when a spell is casted.
    function OnProcessSpell(unit, spell)
    print("OnProcessSpell works")
end

    OnSendChat

    • Called when user send text to the chat.
    function OnSendChat(text)
    print("OnSendChat works - user wrote : " .. text)
end

    OnRecvChat

    • Called when user send recv chat.
    function OnRecvChat(unit, text)
    print("OnRecvChat works - " .. unit.charName .. " wrote : " .. text)
end

    OnSendPacket

    • Called when user send a packet
    function OnSendPacket(packet)
    print("OnSendPacket works")
end

    OnRecvPacket

    • Called when user recv a packet
    function OnRecvPacket(packet)
    print("OnRecvPacket works")
end

    OnBugSplat

    • Called when user bugsplat
    function OnBugSplat()

end

    OnAnimation

    • Called when a unit does an animation
    function OnAnimation(unit, animation)
    print("OnAnimation works: Unit " .. unit.charName .. " did " .. animation)
end

    OnApplyParticle

    • Called when a particle apply to a unit
    function OnApplyParticle(unit, particle)
    print("OnApplyParticle works: Unit " .. unit.charName .. " got " .. particle.charName)
end

    OnNotifyEvent

    • Called on event
    function OnNotifyEvent(event, unit)
    print("OnNotifyEvent " .. unit.charName .. " -> " .. event)
end

    OnNewPath

    • Called on new path
    function OnNewPath(unit, startPos, endPos, isDash, fSpeed, fGravity, fDistance)
    print("OnNewPath works")
end

    OnIssueOrder

    • Called on IssueOrder
    function OnIssueOrder(unit, iAction, targetPos, Target)
    print("OnIssueOrder works")
end

    OnCastSpell

    • Called on CastSpell
    function OnCastSpell(iSpell, startPos, endPos, Target)
    print("OnCastSpell works")
end

    OnApplyBuff

    • Called on ApplyBuff
    function OnApplyBuff(Src, Target, Buff)
    print("OnApplyBuff works")
end

    OnUpdateBuff

    • Called on UpdateBuff
    function OnUpdateBuff(Src, Buff, iStacks)
    print("OnUpdateBuff works")
end

    OnRemoveBuff

    • Called on RemoveBuff
    function OnRemoveBuff(Src, Buff)
    print("OnRemoveBuff works")
end

    Globals Functions

    IsKeyPressed

    • Return true/false if key was pressed

    IsKeyPressed(key)

    IsKeyPressed(string.byte("M"))

    IsKeyDown

    • Return true/false if key is down

    IsKeyDown(key)

    IsKeyDown(string.byte("M"))

    CastSpell

    • Cast a spell

    Uses Spell or Item :

    CastSpell(iSpell)

    Uses Spell or Item at position :

    CastSpell(iSpell, x, z)

    Uses Spell or Item at Target Position :

    CastSpell(iSpell, target)

    LevelSpell

    • Level Up a Spell

    LevelSpell(iSpell)

    PrintChat

    • Print text to chat (local only)

    PrintChat(text)

    SendChat

    • Send Chat (external chat, “/” not handled)

    SendChat(text)

    BlockChat

    • Block last chat message.

    BlockChat()

    DrawText

    • Draw Text over screen

    DrawText(text, size, x, y, ARGB)

    function OnDraw()
    DrawText("this is a draw text", 12, 10, 10, ARGB(255,255,255,255))
end

    DrawLine

    • Draw Line over screen

    DrawLine(startX, startY, endX, endY, size, ARGB)

    function OnDraw()
    DrawLine(10, 10, 50, 10, 2, ARGB(255,255,255,255))
end

    DrawRectangle

    • Draw Rectangle over screen

    DrawRectangle(x, y, width, height, ARGB)

    function OnDraw()
    DrawRectangle(10, 10, 50, 10, ARGB(255,255,255,255))
end

    DrawCircle

    • Draw a circle around game object

    DrawCircle(x, y, z, size, RGB)

    GetTarget

    GetTarget()

    • Return current tick

    GetTickCount()

    GetLatency

    • Return latency

    GetLatency()

    GetCursorPos

    • Return cursor position on the screen

    GetCursorPos()

    createSprite

    • Create a sprite object

    createSprite(file)

    UpdateWindow

    • updates WINDOW_[X/Y/W/H]

    UpdateWindow()

    D3DXVECTOR3

    • Create a D3DXVECTOR3 object
    • D3DXVECTOR3(x, y, z)

    WorldToScreen

    • Return screen position based on ingame coordinates

    WorldToScreen(D3DXVECTOR3)

    Unit Methods

    • Only for Hero

    unit:HoldPosition

    • The unit hold the current position

    unit:HoldPosition()

    unit:MoveTo

    • The unit move to the X and Z position.

    unit:MoveTo(x, z)

    unit:Attack

    • The unit attack the specified target

    unit:Attack(target)

    unit:GetDistance

    • Get the distance from the unit to the target

    unit:GetDistance(target)

    unit:CalcDamage

    • Calculate the damage that can be dealt to the target

    unit:CalcDamage(target, maxDamage)

    unit:CalcMagicDamage

    • Calculate the magic damage that can be dealt to the target

    unit:CalcMagicDamage(target, maxMagicDamage)

    unit:getBuff

    • Return buff object of the specified index

    unit:getBuff(index)

    unit:getInventorySlot

    • Return item ID of the specified slot
    • Slot are from ITEM_1 to ITEM_6

    unit:getInventorySlot(slot)

    unit:getItem

    • Return LoLItem
    • Slot are from ITEM_1 to ITEM_6

    unit:getItem(slot)

    unit:GetSpellData

    • Return Spell object

    unit:GetSpellData(iSpell)

    unit:CanUseSpell

    • Return Spell State

    unit:CanUseSpell(iSpell)
